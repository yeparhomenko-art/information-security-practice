# Electronic Dean's Office
## Опис
REST API на базі FastAPI + SQLite. Практична робота з дисципліни
"Безпека інформаційних систем".
## Запуск
```bash
git clone <https://github.com/yeparhomenko-art/information-security-practice.git>
cd information-security-practice
docker compose up --build
```
## Зупинка
docker compose stop

## Зупинити і видалити контейнери
docker compose down

## Доступ
- API: http://localhost:3010
- Документація (Swagger): http://localhost:3010/docs
